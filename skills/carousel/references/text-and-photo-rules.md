# Правила текста и банка фото (обязательно для carousel/reels/stories)

## 1. Читаемость текста на фото-фоне
При наложении текста на фото/видео ВСЕГДА:
- добавь **тёмную подложку-скрим** (градиент) под текстом;
- высокий контраст (светлый текст на тёмном, не белый на бежевом);
- текст только в пустой зоне, **НИКОГДА не поверх лица**.
Для код-карусели: обложка/финал — фото со скримом, средние слайды — сплошной тёмный фон (гарантированная читаемость).

## 2. Обращение — нейтральное на «ты»
Весь текст — на «ты» в **гендерно-НЕЙТРАЛЬНых** формах (настоящее время), НИКОГДА не женский род.
Правильно: «узнаёшь себя», «всё на тебе». Неправильно: «узнала», «сделала сама». Аудитория смешанная.

## 3. Банк фото + каталог
У пользователя есть личный **банк фото** + `photo-catalog.md` (тип: портрет / в полный рост / селфи, одежда, сцена, назначение). Выбирай фото ПО каталогу, без переспроса:
- обложка/CTA/эксперт → портрет/сцена; вечер/результат → элегантные; сторис/reels → селфи/lifestyle.
- Фото-фон берётся так, чтобы **лицо не обрезалось** (object-position под лицо; для широких слайдов — сплит: фото на половине, текст на другой).
- Избегай кадров, которые пользователь пометил как нежелательные (напр. без макияжа).

## 4. Higgsfield-пул (многоразовый банк фонов)
Дешевле генерить банк из ~10 брендовых фонов с человеком (nano_banana_pro, разные референсы-фото, каждый своего стиля, место под текст) ОДИН раз, а потом собирать карусели КОДОМ поверх них (бесплатно), чем генерить каждый слайд заново.
