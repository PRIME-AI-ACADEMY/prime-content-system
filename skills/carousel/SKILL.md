# Carousel — три маршрута производства

Маршрут (H / C / K) ЗАДАЁТ контент-план в колонке «Формат», а послайдные промпты — в колонке «Higgsfield-промпт / Дизайн». За неделю — три карусели РАЗНЫМИ маршрутами.

## ГЛАВНОЕ ПРАВИЛО
**Higgsfield используется ТОЛЬКО когда на слайде ЧЕЛОВЕК (аватар/фото автора) или сложная сцена.** Генерить «текст на однотонном фоне» через Higgsfield ЗАПРЕЩЕНО — это делает код (C) или Canva (K).

## Маршрут H — Higgsfield (аватар в кадре)
- Модель **`nano_banana_pro`**, aspect_ratio 4:5, resolution 2k. **4 слайда** (обложка + 2 смысловых + финал с CTA).
- **НА КАЖДОМ слайде передавай reference sheet автора** в `medias: [{type:image, role:image, value:<REFERENCE_SHEET_MEDIA_ID>}]` — иначе лицо плывёт от слайда к слайду. Лицо должно быть ОДИНАКОВЫМ на всех слайдах.
- Шаблон промпта слайда: «Use the person from the reference image, keep their exact face. Premium editorial Instagram carousel slide 4:5. <сцена из плана>. Brand palette <цвета design-system>, elegant typography, tasteful graphics, magazine quality. Integrate this EXACT Russian text, crisp and correctly spelled: «<текст слайда>». No watermark, no logos.»
- Проверь текст и лицо; брак — 1 перегенер.

## Маршрут C — Код (Python + Pillow, БЕЗ браузера)
Работает и локально, и в облаке — рисуем холст напрямую.
1. Возьми **реальные фото автора** из Drive-папки `photos` (см. MD-каталог рядом): скачай нужные файлы.
2. Python + Pillow: холст 1080×1350; фото вписать по cover; градиент-скрим брендового цвета (на обложке — СВЕРХУ, чтобы не затемнять лицо); текст шрифтом бренда; плашка CTA на финале; нумерация и ник.
3. Сохрани PNG по порядку и залей в Drive.
**Цепочка запасных:** нет исполнения кода → собери эту карусель маршрутом **K (Canva)**; если есть локальный Chrome — можно HTML→PNG. НИКОГДА не падай — всегда отдавай готовую карусель.

## Маршрут K — Canva
- Через Canva-коннектор: создай дизайн 1080×1350 в бренд-стиле (цвета/шрифты из `design-system`).
- **Фото автора обязательно минимум на обложке и финале** (загрузи из Drive-папки `photos`).
- Экспортируй страницы в PNG и залей в Drive.

## Движок текста (ОБЯЗАТЕЛЬНО)
Перед текстом прочитай **`../content-pipeline/references/copy-engine.md`** + профиль `tone-of-voice`. Без «не X, а Y», без канцелярита, без скобок-рода. Звучит генерично — перепиши.

## Структура текста
Обложка (7–8 слов, хук) → суть одной фразой → одна мысль на слайд → доказательство (цифра/кейс) → финал с CTA (сохрани/перешли или кодовое слово).

## Запрещено
Плоская карусель «текст на однотонном фоне без дизайна»; белый текст на светлом; текст на лице; затемнённое лицо автора; женский род в скобках.

Агент читает `audience`, `offer`, `tone-of-voice`, `design-system`, берёт маршрут и послайдные промпты из плана.
