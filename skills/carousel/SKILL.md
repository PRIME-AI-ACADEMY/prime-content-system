# Carousel — Instagram-карусель (текст + дизайн кодом, merged)

Назначение: собрать карусель целиком — продающий текст слайдов + визуал, свёрстанный КОДОМ (не Canva) по дизайн-системе пользователя. Читает audience, offer, tone-of-voice и design-system. Метод dark-рендера адаптирован из скилла dark-carousel (автор Stanislav Tsys, @stastsis).

## Шаг 0 — контекст и бриф
Прочитай профили. Если не хватает — бриф одним сообщением: ниша/продукт · кто читает · главная боль · цель (охват/доверие/продажа) · тон · цифры/кейсы.

## Шаг 1 — тип карусели (5 типов)
1. **Список** — «[ЧИСЛО] вещей которые [результат]». Охват, сохранения.
2. **Пошаговая инструкция** — «Как [результат] за [шаги]». Доверие.
3. **До/после** — было → что изменила → стало. Продажа.
4. **Мифы и правда** — МИФ → ПРАВДА. Вирусность.
5. **История-кейс** — каждый слайд = поворот. Вовлечение.

## Шаг 2 — текст слайдов
- **Слайд 1 — обложка**: хук, останавливающий скролл. Макс 7–8 слов, конкретика, вызывает «а как?».
- **Слайд 2 — приземление**: сразу СУТЬ одной фразой крупно (НЕ «о чём этот пост», НЕ вопрос). Мелким — обещание структуры.
- **Слайды 3–N**: одна мысль на слайд, 5–8 слов в строке, активные глаголы, «ты».
- **Доказательство**: кейс/цифра.
- **Финал — CTA**: одно действие.
- **Текст под постом**: 2–3 предл. + CTA. Первое слово — НЕ «Я».
Голос — из профиля ToV.

## Шаг 3 — дизайн кодом (принцип «холодного читателя»)
Пиши для человека, который НЕ знает темы: каждый слайд самодостаточен. Правила вёрстки (dark-carousel): слайд 1 = визуал; запретные зоны IG — правый верх/низ; ник — нижний левый, нумерация NN/TT — верхний левый; макс 2 цвета текста; line-height ≥1.0; footer/CTA ≥20px, body ≥22px (не лезет — +1 слайд, лимит IG = 20). Рендер: HTML/CSS → PNG 1080×1350 (по желанию MP4 с видео-фоном). Стиль — из design-system; если её нет — дефолтный editorial + предложи собрать дизайн-систему. Экспорт на Google Drive, ссылка в контент-план. Перед рендером — согласуй структуру.

## Контроль качества
Обложка останавливает скролл? слайд 2 = ответ, не вопрос? один слайд = одна мысль? есть доказательство и CTA? дизайн из design-system? голос из ToV?
