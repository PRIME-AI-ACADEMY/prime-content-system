# Carousel — Instagram-карусель (4 визуальных типа)

Назначение: собрать полную карусель с продающим текстом и визуалом нужного типа.

## Визуальный тип — берётся из плана (колонка Формат) и промптов на каждый слайд
Контент-план задаёт тип и послайдный промпт. Если тип не задан — спроси или выбери по цели.

- **Тип A · Текст-на-фоне:** вёрстка кодом (HTML→headless Chrome, PNG 1080×1350), брендовый фон + типографика. Чисто, быстро. Нужен браузер (локально).
- **Тип B · С фото/аватаром:** каждый слайд — Higgsfield `nano_banana_pro`, `medias` reference = лицо автора; автор в сцене + текст, editorial, бренд-палитра. Работает и в облаке (без браузера). Текст — reads-exactly, кириллица чистая, перегенер брака.
- **Тип C · Скриншоты:** реальные скрины (таблица/Drive/результаты/переписка) через Chrome + подписи кодом. Доверие, build-in-public.
- **Тип D · Кейс-инфографика:** цифры/до-после/сравнения/диаграммы — код или Higgsfield-графика.

Правило: если промпт слайда задаёт аватара — ОБЯЗАТЕЛЬНО передавай референс лица (иначе лицо поплывёт). За период чередуй типы; анализируй «Зашло / Не зашло».

## Движок текста (ОБЯЗАТЕЛЬНО)
Перед написанием текста слайдов прочитай **`../content-pipeline/references/copy-engine.md`** (анти-ИИ голос, банк хуков, CTA-движок). Правила читаемости и фото — `references/text-and-photo-rules.md`. Визуал через Higgsfield — `references/higgsfield-visual-standard.md`.

## 5 типов по СОДЕРЖАНИЮ (ортогонально визуальным)
1. Список — охват и сохранения
2. Пошаговая инструкция — доверие
3. До/после — продажа
4. Мифы и правда — виральность
5. История-кейс — вовлечение

## Структура текста
- **Обложка:** макс 7–8 слов, хук из банка, останавливает скролл.
- **Слайд 2:** суть одной фразой крупно.
- **Слайды 3–N:** одна мысль на слайд, 40–60 слов.
- **Доказательство:** кейс или цифра обязательны.
- **Финал (CTA на рост):** полезная → «сохрани + перешли»; продающая → слово в директ.

## Вёрстка (тип A/C/D кодом)
- PNG 1080×1350 (или MP4). Максимум 2 цвета текста на слайд.
- Мин 20px footer, 22px body. Ник — нижний левый угол, нумерация NN/TT — верхний левый.
- Цвета/шрифты/сетка — строго по `design-system`.

## Запрещено
Водянистые вступления, пассивный залог, абстракции, стоп-слова, белый текст на светлом, текст на лице, женский род. **Запрещена плоская карусель «текст на однотонном фоне без дизайна»** — даже тип A делай богатым (текстура/градиент/геометрия/крупные акценты).

Агент читает профили `audience`, `offer`, `tone-of-voice`, `design-system`, берёт тип и послайдные промпты из плана, прогоняет текст через самопроверку движка.
