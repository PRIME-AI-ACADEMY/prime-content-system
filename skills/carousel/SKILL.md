# Carousel — Instagram-карусель (текст + дизайн кодом)

Назначение: собрать полную карусель с продающим текстом и визуалом, свёрстанным кодом.

## Движок текста (ОБЯЗАТЕЛЬНО)
Перед написанием текста слайдов прочитай **`../content-pipeline/references/copy-engine.md`** (анти-ИИ голос, банк хуков, CTA-движок). Правила читаемости и фото — `references/text-and-photo-rules.md`. Визуал через Higgsfield — `references/higgsfield-visual-standard.md`.

## 5 типов карусели
1. Список — охват и сохранения
2. Пошаговая инструкция — доверие
3. До/после — продажа
4. Мифы и правда — виральность
5. История-кейс — вовлечение

## Структура текста
- **Обложка:** макс 7–8 слов, хук из банка, останавливает скролл.
- **Слайд 2:** суть одной фразой крупно.
- **Слайды 3–N:** одна мысль на слайд, 40–60 слов.
- **Доказательство:** кейс или цифра обязательны.
- **Финал (CTA на рост):** полезная карусель → «сохрани + перешли другу»; продающая → слово в директ (CTA-движок).

## Вёрстка кодом
- PNG 1080×1350 (или MP4). Максимум 2 цвета текста на слайд.
- Мин 20px для footer, 22px для body. Ник — нижний левый угол, нумерация NN/TT — верхний левый.
- Цвета/шрифты/сетка — строго по профилю `design-system`.

## Запрещено
Водянистые вступления, пассивный залог, абстракции, стоп-слова из движка («как мы все знаем» и пр.), белый текст на светлом фоне, текст на лице, женский род.

Агент читает профили `audience`, `offer`, `tone-of-voice`, `design-system`, прогоняет текст через самопроверку движка и согласует структуру перед рендером.
