# Long Post — развёрнутые посты

Назначение: пост на 2000–3000 знаков, который читают до конца и сохраняют.

## ОБЯЗАТЕЛЬНО перед генерацией
Прочитай **`../content-pipeline/references/copy-engine.md`** (анти-ИИ голос, хуки, паттерны, каркасы, CTA). Прочитай `audience`, `offer`, `tone-of-voice`, `hunt-ladder` (под ступень осознанности).

## 5 структур (выбери под тему)
1. **Трансформация:** было → переломный момент → стало → урок. Лично, с деталями.
2. **Разбор механики:** как что-то работает по шагам, с примерами.
3. **Разрушение стереотипа:** «все думают X — на самом деле Y» (каркас «Разрушитель мифов»).
4. **Рефлексия-итог:** мысль/вывод из опыта, эмоция + инсайт.
5. **Разбор тренда:** что меняется в нише и что с этим делать.

## Правила
- **Сильное начало:** первые 2 строки — хук из банка, без разгона.
- Один абзац = одна мысль + воздух. Конкретика и эмоция, не канцелярит.
- Соц. доказательство (кейс/цифра) хотя бы одно.
- **Финал — CTA на рост:** вывод + одно действие (сохрани/перешли/вопрос в комменты/слово в директ — по цели).

## Выдача
Готовый пост + 2 альтернативных первых строки (для A/B). Прогони через самопроверку движка.
