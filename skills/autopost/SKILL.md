# Autopost — автопостинг через Postiz

Назначение: опубликовать/запланировать готовый контент из контент-плана в соцсети. Работает с любыми каналами Postiz (Instagram, Facebook, TikTok, LinkedIn, Threads и т.д.), не только Instagram.

## Вход
Строка контент-плана (Google-таблица): текст поста, ссылка на медиа в Google Drive, статус, дата.

## Шаги
1. Прочитай контент-план (Composio GOOGLESHEETS_BATCH_GET).
2. Отбери строки: статус = «Запланировано» (или «Утверждено», если включён ручной апрув), есть ссылка на медиа, отметка Postiz пустая.
3. Получи файлы из папки Drive (GOOGLEDRIVE_FIND_FILE → webContentLink).
4. Загрузи медиа в Postiz: `uploadFromUrlTool`.
5. Запланируй пост: `integrationSchedulePostTool` — канал (integrationId из `integrationList`), текст = подпись из плана, дата из плана, type = "schedule" (или "draft" для ручного контроля).
6. Обнови таблицу: статус = «Опубликовано» + отметка Postiz.

## Правила
- Публикуй ТОЛЬКО строки, явно готовые к публикации. Не падай с ошибкой, если постить нечего — это норма.
- Один пост — один канал (для мульти-канала повтори по integrationId).
- Подпись — из голоса бренда (ToV), нейтральное обращение на «ты».

Коннекторы: Postiz, Composio (Google Sheets + Drive).
