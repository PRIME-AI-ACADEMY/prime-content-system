# Tone of Voice Builder — конструктор голоса бренда

Назначение: извлечь голос пользователя из его текстов и сохранить как **профиль ToV** — источник правды для всех контент-скиллов. Голос нельзя выдумать — его вытаскивают из примеров.

## Шаги
1. **Образцы**: попроси 3–5 текстов пользователя (посты, сторис, расшифровки голосовых). Без 3 образцов точный анализ невозможен.
2. **Разобрать голос**: тон; обращение (ты/вы); ритм фраз; лексика; эмоции/усиление; структура (хук→мысль→CTA); эмодзи; запреты.
3. **Списки слов**: любимые и запрещённые (по умолчанию: «является»→«это», «позволяет»→«можно/даёт», «данный»→«этот», «в рамках»→убрать).
4. **Собрать профиль** + пример «до/после» (перепиши нейтральную фразу голосом пользователя).
5. **Сохранить** в Google Drive («Tone of Voice»); иначе `profile/tone-of-voice.md`.

## Шаблон
```markdown
# ПРОФИЛЬ TONE OF VOICE
Тон: ... / Обращение: ты/вы / Ритм: ...
Лексика и фишки: ... / Эмоции: ... / Эмодзи: ...
Любимые слова: ... / Запрещённые слова: ...
Пример до/после: ...
```

## Контроль качества
Профиль из РЕАЛЬНЫХ образцов? Пример «до/после» звучит как пользователь? Есть списки любимых/запрещённых слов? Если образцов < 3 — собери ещё.
