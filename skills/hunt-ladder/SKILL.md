# Hunt Ladder — лестница узнавания Бена Ханта

Назначение: разложить аудиторию по 6 уровням осознанности и для каждого определить, какой контент её двигает вверх. Мост между audience-builder и content-plan. Читает профиль аудитории.

## 6 ступеней
0. **Нет проблемы** — вскрыть проблему.
1. **Есть проблема** — усилить боль, цена бездействия.
2. **Ищет решения** — обзор способов, почему твой метод.
3. **Сравнивает решения** — твой метод vs альтернативы, кейсы.
4. **Сравнивает тебя** — отзывы, демо, оффер, отстройка.
5. **Готов купить** — оффер, дедлайн, гарантия, прямой CTA.

## Шаги
1. Прочитай профиль аудитории, размести по ступеням (укажи % примерно).
2. Для каждой занятой ступени — 2–3 сообщения языком клиента + форматы.
3. Сохрани «Лестница Ханта» рядом с профилем аудитории; иначе `profile/hunt-ladder.md`. Вывод: где прогрев, где продажа.
