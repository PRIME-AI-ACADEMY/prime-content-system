# Content Routine — ежедневный автопилот

Две задачи по расписанию, которые превращают контент-план в автономный конвейер.

## Утро (напр. 08:00) — генерация на N дней вперёд
1. Прочитай контент-план. Отбери строки на ближайшие N дней без медиа (ссылка пустая).
2. Для каждой сгенерируй контент по её формату (carousel / reels / stories …) — используй скиллы `carousel`, `reels` и т.д., дизайн-систему и банк фото.
3. Залей в новую папку Google Drive → впиши ссылку, статус = «Запланировано».
4. Ничего не публикуй.

## Ночь (00:00) — автопостинг + проверка
1. Прочитай план. Если все строки со ссылкой имеют статус «Запланировано»/«Опубликовано» — всё ок (А НЕ ошибка).
2. Для строк «Запланировано» с медиа → скилл `autopost` (Postiz).
3. Отметь «Опубликовано».

## Как поставить
Cron-задачи (локально `~/.claude/scheduled-tasks/` или облачная рутина). Промпт каждой задачи — самодостаточный (id таблицы, коннекторы, правила). Человеческий контроль = опционально статус «Утверждено».

Коннекторы: Higgsfield, Composio (Drive/Sheets), Postiz.
