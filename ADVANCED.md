## 1. HyperFrames — анимированные видео кодом

Что это: фреймворк, которым Клод собирает видео из кода: анимированные титры и заголовки, субтитры под озвучку, переходы между сценами, анимация под музыку. Идеально для: анимированных обложек, текстовых рилсов, титров поверх видео.

⚠️ Не путать с HeyGen (сервис аватаров) — это разные вещи.

### Установка (один раз)
1. **Node.js** (если ещё нет): Mac — в Терминале `brew install node` · Windows — в PowerShell `winget install OpenJS.NodeJS.LTS`
2. **Плагин** — в Claude Code:
```
/plugin install hyperframes@claude-plugins-official
```
3. Перезапусти приложение Claude.

### Как пользоваться
Просто проси словами, Клод сам возьмёт нужный скилл:
- «Сделай анимированную обложку 9:16 в моей дизайн-системе: заголовок появляется слово за словом»
- «Добавь к этому видео субтитры по моей озвучке»
- «Собери текстовый рилс на 30 секунд из этих 5 тезисов с анимацией и переходами»
Рендер идёт на твоём ноутбуке (как и карусели).

## 2. Canva — для тех, кто привык к Canva

У Claude есть официальный коннектор Canva: Клод создаёт дизайны прямо в твоём Canva-аккаунте (в том числе из бренд-шаблонов) и экспортирует готовые файлы.

### Подключение
Настройки Claude → **Connectors** → **Browse connectors** → найди **Canva** → Connect → войди в свой Canva-аккаунт и разреши доступ.

✅ Проверка: «покажи мои последние дизайны в Canva».

### Когда что выбирать
| | Карусели кодом (база системы) | Canva-коннектор |
|---|---|---|
| Точность бренда | пиксель в пиксель по дизайн-системе | зависит от шаблонов |
| Где работает | ноутбук (Chrome) | облако — даже с телефона |
| Ручная доводка | правки через Клода | привычный редактор Canva |
| Автопилот/рутина | да | частично |

Рекомендация: основной поток — кодом (полная автоматизация), Canva — когда нужна ручная доводка или работа с телефона.
