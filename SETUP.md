# Контент-завод на Claude · Пошаговая установка
**Воркшоп PRIME AI**

Что получится: своя система, которая по одному промту готовит контент на неделю — тексты, карусели, видео — складывает всё в твою Google-таблицу и постит по расписанию.

---

## Часть 0. Что нужно ДО начала

- [ ] **Ноутбук Mac или Windows** — только для настройки (см. врезку ниже)
- [ ] Установлен **Google Chrome**
- [ ] Подписка **Claude Pro** — [claude.ai](https://claude.ai)
- [ ] Аккаунт **Google** (Drive + Таблицы)
- [ ] Аккаунт **higgsfield.ai** (план Plus)
- [ ] **Instagram** (бизнес/креатор)
- [ ] Материалы: 30+ фото, описание бизнеса, 3–5 своих текстов, любимые цвета/шрифты

### 💡 Нет своего ноутбука? Одолжи на день — это работает
Всё ценное живёт **в облаке и в твоих аккаунтах**, а не в ноутбуке: чаты и коннекторы — в аккаунте Claude, контент и таблица — в твоём Google Drive, генерации — в Higgsfield, расписание — в Postiz, сама система — на GitHub.

На чужом ноутбуке: заходи в **свои** аккаунты → делай настройку → в конце дня **выйди из всех аккаунтов** и удали папку проекта. Когда появится свой ноутбук — повтори части 1–4 (≈20 минут), остальное подтянется из облака.

После настройки ежедневная работа завода идёт в облаке, утверждать контент можно с телефона.

## Часть 1. Открываем терминал (1 мин)

- **Mac:** `Cmd + Пробел` → набери «Терминал» → Enter
- **Windows:** Пуск → набери «PowerShell» → Enter

## Часть 2. Инструменты (10 мин)

**Google Chrome** — [google.com/chrome](https://google.com/chrome). Он рендерит твои карусели.

**ffmpeg** — монтаж видео (карусели работают и без него):

- **Mac** (Терминал, две команды):
  ```
  /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  ```
  ```
  brew install ffmpeg
  ```
- **Windows** (PowerShell):
  ```
  winget install ffmpeg
  ```

Отдельные программы для дизайна и моушена НЕ нужны: карусели рендерит Chrome, монтаж — ffmpeg, анимации и видео генерит Higgsfield в облаке.

## Часть 3. Приложение Claude (10 мин)

1. Скачай **Claude Desktop**: [claude.ai/download](https://claude.ai/download) (Mac и Windows)
2. Войди с Pro-аккаунтом
3. Открой раздел **Code**, создай папку проекта `Мой контент-завод`
4. Внутри создай папку `photos` и положи свои фотографии

## Часть 4. Плагин (5 мин)

В строке Claude Code по очереди:

```
/plugin marketplace add PRIME-AI-ACADEMY/prime-content-system
```
```
/plugin install prime-content-system@prime-ai
```

Перезапусти приложение. Обновление: `/plugin marketplace update prime-ai`

## Часть 5. Разрешения (5 мин)

Скажи Клоду в чате:

```
Создай в папке проекта файл .claude/settings.local.json с разрешениями:
Bash(ffmpeg:*), Bash(ffprobe:*), Bash(curl:*), Bash(python3:*), Bash(bash:*),
Bash(mkdir:*), Bash(cp:*), Bash(ls:*), Bash(find:*), Bash(sips:*),
и путь к Google Chrome для рендера каруселей (найди его на этой системе сам).
```

Когда Клод спрашивает разрешение — выбирай **«Always allow»**.

## Часть 6. Коннекторы (15–20 мин)

Настройки Claude → **Connectors**. Привязаны к твоему аккаунту, не к ноутбуку.

**На воркшопе:** 1. **Higgsfield** — на higgsfield.ai включи подключение к Claude (MCP).

**Дома:** 2. **Composio** (Google Drive + Sheets) — логин через Google. 3. **Postiz** — postiz.com → подключи Instagram → включи MCP.

## Часть 7. Запуск: фундамент (30–40 мин)

Открой НОВЫЙ чат в Claude Code и вставь целиком:

```
Настраиваем мой контент-завод с нуля по плагину prime-content-system.

1. Задай мне короткий бриф: ниша, цель в числах, продукт/оффер, аудитория, тон.
2. Собери фундамент по порядку: business-dna → audience-builder → hunt-ladder →
   offer-builder → tone-of-voice-builder → design-system-builder
   (обязательно спроси мои любимые цвета и шрифты или предложи 2–3 палитры).
   Каждый блок показывай мне на утверждение.
3. Мои фото лежат в папке photos — составь photo-catalog.md:
   раздели по типу (портрет / полный рост / селфи), опиши образ и назначение.
4. Покажи ПРЕДВАРИТЕЛЬНЫЙ контент-план на неделю прямо в чате
   (Дата · Формат · Тема · Идея · Хук · Ступень · CTA). В таблицу пока не пиши.
5. После моего «утверждаю» — распиши ОБРАЗЕЦ: 2 поста разных форматов
   (карусель + рилс) полностью, со всеми промтами. Я проверю стиль.
6. После одобрения образца — распиши остальные строки и собери красивую
   Google-таблицу (14 колонок, шапка в моём фирменном цвете, статус-дропдаун).
7. Сгенерируй 1 карусель по первой строке: текст кодом по моей дизайн-системе,
   фото из каталога, рендер через Chrome, залей в Google Drive,
   ссылку и статус «На утверждении» — в таблицу.

Текст пиши живым языком по copy-engine: без «не X, а Y», без скобок-рода,
обращение на «ты», CTA на рост (сохрани / перешли / пиши кодовое слово).
```

## Часть 8. Что в облаке, а что на ноутбуке

| В облаке (работает без ноутбука) | На ноутбуке |
|---|---|
| Тексты, планы, сценарии (Claude) | Рендер каруселей кодом (Chrome) |
| Картинки и видео (Higgsfield) | Монтаж видео (ffmpeg) |
| Таблица и файлы (Google) | Папка с фото* |
| Автопостинг (Postiz) | |

\* Совет: попроси Клода залить банк фото в Google Drive — тогда и фото будут в облаке.

**Автопилот (рутина):** после 3–4 дней ручной обкатки система ставится на расписание — утром генерирует контент, ночью постит утверждённое. Инструкция — отдельно после воркшопа.

## Часть 9. Ежедневная работа

- **Новый контент:** «Сделай карусель на [дата] из плана» / «Добавь следующую неделю»
- **Утверждение:** таблица (можно с телефона) → статус «Утверждено»
- **Автопостинг:** «Проверь таблицу и запланируй утверждённое в Postiz»

## Частые проблемы

| Симптом | Решение |
|---|---|
| Клод спрашивает разрешение | «Always allow» — один раз на инструмент |
| Карусель не рендерится | Проверь Chrome; скажи «найди Chrome на этой системе и отрендери снова» |
| Плагин не виден | Перезапусти приложение Claude |
| winget/brew не найдены | Windows: обнови «Установщик приложений» из Microsoft Store · Mac: поставь Homebrew (часть 2) |
| Кончились лимиты Pro | Подождать сброс или Max-подписка |
| Текст «звучит как ИИ» | «Прогони через аудит copy-engine и перепиши» |
