# Контент-завод на Claude · Установка с нуля, шаг за шагом
**Воркшоп PRIME AI**

Результат: твоя система, которая по одному промту готовит контент на неделю (тексты, карусели, видео), складывает в твою Google-таблицу и публикует по расписанию.

**Как пользоваться гайдом:** иди строго по шагам, после каждого шага есть ✅ Проверка. Если что-то пошло не так — скопируй текст ошибки и спроси у Клода, он чинит сам.

---

## 📖 Словарик (30 секунд)

- **Claude Code** — режим Claude, где он умеет работать с файлами и запускать программы. Наш «цех».
- **Терминал** — окно, куда вводят текстовые команды. Нужен 2 раза за весь гайд.
- **Плагин** — набор навыков (скиллов) контент-завода, ставится двумя командами.
- **Коннектор (MCP)** — «провод» между Claude и сервисом (Higgsfield, Google, Postiz). Подключается кнопками, привязан к твоему аккаунту.

---

## Шаг 1. Ноутбук

Нужен **Mac или Windows** — только для настройки и рендера каруселей. Управлять готовой системой можно с телефона.

### 💡 Нет своего ноутбука — одолжи на день, это работает
Всё ценное живёт в твоих аккаунтах (Claude, Google, Higgsfield, Postiz) и на GitHub — не в ноутбуке. На чужом ноуте: входишь в **свои** аккаунты → настраиваешь → в конце дня выходишь из всех аккаунтов и удаляешь папку проекта. На своём будущем ноуте повторишь шаги 3–6 за 20 минут — остальное подтянется из облака.

## Шаг 2. Регистрируем аккаунты (30 мин, можно до воркшопа)

2.1. **Claude Pro** → [claude.ai](https://claude.ai) → Sign up → в настройках аккаунта **Upgrade → план Pro** (~20 $/мес)
2.2. **Google** → обычный Gmail-аккаунт (Drive и Таблицы уже внутри)
2.3. **Higgsfield** → [higgsfield.ai](https://higgsfield.ai) → Sign up → тариф **Plus** (генерация фото/видео)
2.4. **Postiz** (автопостинг; можно после воркшопа) → [postiz.com](https://postiz.com) → Sign up. *Бесплатная альтернатива — Meta Business Suite, см. Шаг 11.*
2.5. **Instagram** — переключи на профессиональный аккаунт (Настройки → Тип аккаунта → Креатор/Бизнес)

✅ Проверка: можешь войти на все 4 сайта.

## Шаг 3. Устанавливаем программы (20 мин)

### 3.1. Google Chrome
[google.com/chrome](https://google.com/chrome) → Скачать → установить. Chrome рендерит твои карусели в картинки.

### 3.2. Приложение Claude
[claude.ai/download](https://claude.ai/download) → скачай для своей системы → установи → войди в свой Pro-аккаунт.

### 3.3. ffmpeg (монтаж видео)

**Mac:**
1. Открой Терминал: `Cmd + Пробел`, набери `Терминал`, Enter.
2. Вставь и нажми Enter (установка Homebrew):
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
   Спросит пароль от Мака — вводи (символы не отображаются, это нормально), Enter.
3. Затем вторая команда:
```
brew install ffmpeg
```

**Windows:**
1. Пуск → набери `PowerShell` → Enter.
2. Вставь и нажми Enter:
```
winget install ffmpeg
```
   Если «winget не найден» — открой Microsoft Store, обнови «App Installer», повтори.

✅ Проверка: `ffmpeg -version` → строки с версией.

### 3.4. Голосовой ввод (очень рекомендую)
Печатать промты голосом в 3–4 раза быстрее.
- **Wispr Flow** — [wisprflow.ai](https://wisprflow.ai), Mac и Windows: горячая клавиша → говоришь → текст в любом поле. Есть бесплатный план.
- Бесплатно встроенное: **Mac** — Настройки → Клавиатура → Диктовка (вкл), вызов двойным нажатием 🎤/Fn. **Windows** — `Win + H`.

### 3.5. Что ставить НЕ нужно
Никаких «программ для дизайна и моушена»: карусели рендерит Chrome, монтаж — ffmpeg, картинки/видео/анимации генерит Higgsfield в облаке.

## Шаг 4. Папка проекта (5 мин)

1. Создай папку `Контент-завод`: **Mac** — Документы → правый клик → Новая папка. **Windows** — Документы → правый клик → Создать → Папку.
2. Внутри создай папку `photos` и скопируй туда 30+ своих фото. *(Вариант «фото в Google Drive» — Шаг 8.)*
3. Приложение Claude → раздел **Code** → **Open folder / Открыть папку** → выбери `Контент-завод`.

✅ Проверка: в окне Code видно имя папки `Контент-завод`.

## Шаг 5. Плагин (5 мин)

В строке ввода Claude Code набери (именно со слэша) и отправь:
```
/plugin marketplace add PRIME-AI-ACADEMY/prime-content-system
```
Дождись подтверждения, затем:
```
/plugin install prime-content-system@prime-ai
```
Полностью закрой приложение Claude и открой снова.

✅ Проверка: спроси Клода «какие скиллы prime-content-system тебе доступны?» — перечислит 16.

Обновление в будущем: `/plugin marketplace update prime-ai`

## Шаг 6. Разрешения (5 мин)

Скажи Клоду в чате (можно голосом):
```
Создай в папке проекта файл .claude/settings.local.json с разрешениями:
Bash(ffmpeg:*), Bash(ffprobe:*), Bash(curl:*), Bash(python3:*), Bash(bash:*),
Bash(mkdir:*), Bash(cp:*), Bash(ls:*), Bash(find:*), Bash(sips:*),
и добавь путь к Google Chrome на этой системе для рендера каруселей — найди его сам.
```
Дальше правило одно: когда Клод спрашивает разрешение — жми **«Always allow»**.

## Шаг 7. Коннекторы — подключаем сервисы к Claude (30 мин)

Где: приложение Claude → **Настройки (шестерёнка) → Connectors**. Два пути:
- **Browse connectors** — галерея готовых: нажал → Connect → авторизовался.
- **Add custom connector** — подключение по ссылке: Name + URL → Add → Connect.

⚠️ Интерфейсы обновляются. Если кнопка называется иначе — ищи слова «MCP», «Integrations», «API» или спроси Клода.

### 7.1. Higgsfield — нужен на воркшопе
1. Войди на [higgsfield.ai](https://higgsfield.ai).
2. Проверь галерею: Connectors → **Browse connectors** → поищи «Higgsfield» → Connect → авторизуйся.
3. Если в галерее нет: на higgsfield.ai открой раздел **MCP / Integrations** (меню профиля или низ сайта) → скопируй ссылку → Claude: **Add custom connector** → Name `Higgsfield`, URL → Add → **Connect** → войди и разреши доступ.

✅ Проверка: «какие инструменты Higgsfield тебе доступны?» → generate_image, generate_video…

### 7.2. Composio: Google Drive + Таблицы — можно дома
1. [mcp.composio.dev](https://mcp.composio.dev) → Sign up (через Google).
2. Создай MCP-сервер, включи в нём **Google Sheets** и **Google Drive**.
3. Скопируй выданный URL.
4. Claude → Connectors → **Add custom connector** → Name `Composio`, URL → Add → Connect.
5. При первой команде Клод пришлёт ссылку авторизации Google → открой → выбери свой аккаунт → **Разрешить**.

✅ Проверка: «создай тестовую Google-таблицу и дай ссылку».

### 7.3. Postiz — можно дома
1. [app.postiz.com](https://app.postiz.com) → **Add channel** → Instagram → авторизуйся.
2. В настройках Postiz найди **MCP / API** → скопируй ссылку MCP.
3. Claude → Connectors → **Add custom connector** → Name `Postiz`, URL → Add → Connect.

✅ Проверка: «покажи мои каналы в Postiz».

## Шаг 8. Фото — два варианта хранения

**A. Папка на ноутбуке** (проще, старт за минуту): фото уже в `photos` — готово.
**B. Google Drive** (нужно для полного автопилота): создай в Drive папку `Контент-завод/Фото`, загрузи фото, скажи Клоду: «мои фото в папке Drive [ссылка], составь каталог».

Рекомендация: начни с A, потом скажи Клоду: **«залей мой фото-банк в Google Drive и дальше используй облачную версию»**.

## Шаг 9. Запуск фундамента (30–40 мин)

Открой НОВЫЙ чат в Claude Code и вставь целиком:

```
Настраиваем мой контент-завод с нуля по плагину prime-content-system.

1. Задай мне короткий бриф: ниша, цель в числах, продукт/оффер, аудитория, тон.
2. Собери фундамент по порядку: business-dna → audience-builder → hunt-ladder →
   offer-builder → tone-of-voice-builder → design-system-builder
   (обязательно спроси мои любимые цвета и шрифты или предложи 2–3 палитры).
   Каждый блок показывай мне на утверждение.
3. Мои фото лежат в папке photos — составь photo-catalog.md:
   раздели по типу (портрет / полный рост / селфи), опиши образ и назначение.
4. Покажи ПРЕДВАРИТЕЛЬНЫЙ контент-план на неделю прямо в чате
   (Дата · Формат · Тема · Идея · Хук · Ступень · CTA). В таблицу пока не пиши.
5. После моего «утверждаю» — распиши ОБРАЗЕЦ: 2 поста разных форматов
   (карусель + рилс) полностью, со всеми промтами. Я проверю стиль.
6. После одобрения образца — распиши остальные строки и собери красивую
   Google-таблицу (14 колонок, шапка в моём фирменном цвете, статус-дропдаун).
7. Сгенерируй 1 карусель по первой строке: текст кодом по моей дизайн-системе,
   фото из каталога, рендер через Chrome, залей в Google Drive,
   ссылку и статус «На утверждении» — в таблицу.

Текст пиши живым языком по copy-engine: без «не X, а Y», без скобок-рода,
обращение на «ты», CTA на рост (сохрани / перешли / пиши кодовое слово).
```

✅ Проверка дня: в твоём Google Drive — готовая карусель, в таблице — план на неделю.

## Шаг 10. Телефон: что можно, что нельзя

Поставь приложение **Claude** на телефон (iOS/Android), войди в тот же аккаунт.

**Можно с телефона:** тексты и идеи, генерации Higgsfield (облако), облачные Code-сессии, утверждение в приложении Google Таблицы (статус «Утверждено»), ответы на «СИСТЕМА» в директе.
**Нельзя с телефона:** установить плагин, отрендерить карусель кодом (нужен Chrome на компьютере).
Формула: **завод строим на ноутбуке — управляем с телефона.**

## Шаг 11. Автопостинг: три варианта

| Вариант | Цена | Что умеет |
|---|---|---|
| **Postiz** (рекомендуем) | облачный платный; self-hosted бесплатный | Полный автопилот: Клод сам ставит посты в расписание через коннектор |
| **Meta Business Suite** — [business.facebook.com](https://business.facebook.com) | бесплатно | Планировщик Instagram+Facebook. Клод готовит контент и подписи → ты за 2 минуты ставишь в планировщик. Полуавтомат |
| **Buffer / Metricool** (free) | бесплатно с лимитами | Планировщик с ограничениями |

Честно: настоящий «сам публикует» — только с Postiz (у него единственного есть MCP). Остальное = «система готовит, ты нажимаешь опубликовать».

## Шаг 12. Автопилот (рутина)

Когда 3–4 дня поработаешь руками и утвердишь стиль — система ставится на расписание: утром генерирует контент, ночью публикует утверждённое. Условия: фото-банк в Google Drive (Шаг 8B) + Postiz. Инструкция — отдельно после воркшопа.

## Частые проблемы

| Симптом | Решение |
|---|---|
| Клод спрашивает разрешение | «Always allow» — один раз на инструмент |
| Коннектор не подключается | Перепроверь URL; ищи на сайте сервиса «MCP»/«Integrations»; спроси Клода |
| Карусель не рендерится | «Найди Chrome на этой системе и отрендери снова» |
| Плагин не виден | Полностью перезапусти приложение Claude |
| winget/brew не найдены | Windows: обнови «App Installer» в Microsoft Store · Mac: поставь Homebrew (Шаг 3.3) |
| Кончились лимиты Pro | Подождать сброс окна или Max-подписка |
| Текст «звучит как ИИ» | «Прогони через аудит copy-engine и перепиши» |
| Всё сломалось, паника | Закрой приложение, открой заново, опиши Клоду словами, что видишь |
